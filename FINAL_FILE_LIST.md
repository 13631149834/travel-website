## 部署说明

### 部署方式

#### 方式一：静态部署 (推荐)
将所有HTML、CSS、JS文件上传到任意静态服务器：

```bash
# 使用任意静态服务器
# Nginx / Apache / Vercel / Netlify / GitHub Pages
```

#### 方式二：使用Supabase后端
1. 创建Supabase项目
2. 导入 `supabase-schema.sql`
3. 配置 `js/supabase-config.js` 中的连接信息
