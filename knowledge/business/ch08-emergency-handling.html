第8章 导游应变技能

一、个别要求的处理

★1. 处理个别要求的基本原则

（1）合理而可能原则

对游客提出的要求，导游应首先判断：

  • 要求是否合理
  • 要求是否可能实现

合理且可能的，应积极努力满足；不合理或不可能的，应耐心解释。

（2）认真倾听、耐心解释原则

  • 认真倾听游客诉求
  • 不立即否定
  • 耐心解释原因

（3）尊重游客、不卑不亢原则

  • 尊重人格
  • 维护尊严
  • 不有求必应
常出单选题：处理游客个别要求时，首先要判断"合理而可能"，只有合理且可能的要求才应满足。

★2. 餐饮个别要求的处理

（1）特殊的饮食要求

  • 要求换餐厅：了解原因，尽量满足
  • 要求单独用餐：可同意但费用自理
  • 要求提供风味餐：与旅行社联系，费用自理

（2）宗教饮食禁忌

  • 必须提前告知餐厅
  • 严格遵守禁忌
  • 如误供禁忌食物，应立即道歉并更换

3. 住房个别要求的处理

  • 要求调换房间：查明原因，尽量满足
  • 要求更高标准房：若有可满足，费用自理
  • 要求购买客房：可协助，费用自理
  • 要求单独住一间：住房费自理

4. 交通个别要求的处理

  • 要求改变交通工具：一般不应答应
  • 要求换乘下一航班：应请游客自行办理，导游可协助

5. 游览个别要求的处理

  • 要求更换景点：不得擅自答应，应报告旅行社
  • 要求增加景点：与旅行社联系，重新计算费用
  • 要求减少景点：若不影响行程，可同意
  • 要求自由活动：在不影响团队行程时可同意，但需交代注意事项
常出多选题：游客要求自由活动时，导游应提醒安全、告知联系方式、约定集合时间地点。

6. 购物个别要求的处理

  • 要求退还商品：协助办理，7日内可退
  • 要求代为购买：谨慎对待，事后纠纷多
  • 要求单独外出购物：可提供信息，但一般不陪同

7. 导游个别要求的处理

  • 要求中途退出：必须经旅行社同意，妥善安排交接
  • 要求更换导游：报告旅行社，耐心解释

二、问题与事故的处理

★1. 旅游计划变更的处理

（1）缩短行程（漏游）

  • 立即报告旅行社
  • 制定应变计划
  • 向游客耐心解释
  • 给予适当补偿

（2）延长行程（漏游）

  • 及时报告旅行社
  • 重新安排食宿交通
  • 办理相关手续
  • 增加的费用由游客承担

（3）取消行程

  • 妥善安排游客滞留期间的食宿
  • 安排返程交通
  • 费用问题按合同约定处理

★2. 误机（车、船）事故的处理

（1）预防措施

  • 提前确认交通票据
  • 国内航班：提前2小时到达机场
  • 国际航班：提前3小时到达机场
  • 火车/轮船：提前1小时到达车站/码头
★ 误机（车、船）预防时间表：
国内航班提前2小时
国际航班提前3小时
火车/轮船提前1小时

（2）应急处理程序

一旦发生误机事故，应立即：

  1. 立即报告旅行社
  2. 安排游客尽快离港
  3. 联系航空公司争取改签
  4. 安排好游客食宿
  5. 承担相关费用（因导游责任）
常出单选题：误机事故是导游的重大责任事故，一旦发生必须立即报告旅行社并采取补救措施。

3. 游客走失的处理

（1）预防措施

  • 做好安全提醒
  • 多次清点人数
  • 明确集合时间和地点
  • 发放联系方式卡片

（2）走失处理程序

  • 立即报告旅行社
  • 组织寻找
  • 联系景区广播
  • 若找不到，报告景区管理部门或报警
  • 事后分析原因，总结教训

4. 游客患病或突发疾病的处理

（1）预防措施

  • 了解游客健康状况
  • 注意休息和饮食
  • 提醒携带常用药品

（2）处理原则

  • 立即采取措施急救
  • 尽快送医院
  • 通知旅行社
  • 让患者的亲属或领队到场

（3）费用处理

  • 医疗费用由游客自理
  • 导游陪同费用由旅行社承担

5. 游客死亡事故的处理

  • 立即报告旅行社和当地公安机关
  • 保护好现场
  • 开具死亡证明
  • 配合做好善后工作
  • 遗体处理应征得亲属同意

6. 交通事故的处理

（1）立即处理

  • 立即报警
  • 保护现场
  • 组织抢救伤员
  • 报告旅行社

（2）善后处理

  • 配合交警处理
  • 做好安抚工作
  • 做好取证工作

7. 火灾事故的处理

  • 立即拨打119
  • 迅速疏散游客
  • 使用灭火器灭火
  • 自救逃生时压低身体
  • 切勿乘电梯

8. 食物中毒的处理

  • 立即催吐（中毒2小时内）
  • 尽快送医院
  • 保留食物样本
  • 报告旅行社和卫生防疫部门
📌 口诀记忆：突发事故莫慌张，沉着冷静是首项；误机误车立即报，游客走失搜寻忙；疾病死亡送医院，火灾逃生不乘梯；交通事故先报警，食物中毒催吐先。

三、自然灾害的避险

★1. 地震避险

（1）室内避险

  • 迅速撤离到空旷地带
  • 来不及撤离时，躲在坚固家具下
  • 保护头部
  • 远离玻璃窗和悬挂物

（2）室外避险

  • 远离建筑物和电线杆
  • 在空旷处蹲下
  • 保护头部

（3）导游职责

  • 镇定组织疏散
  • 清点人数
  • 安抚游客情绪
  • 报告旅行社

2. 洪水避险

  • 迅速向高处转移
  • 不要靠近河流边
  • 被困时寻找漂浮物
  • 被洪水卷住时抓住漂浮物等待救援

3. 台风避险

  • 关注天气预报
  • 进入坚固建筑物躲避
  • 远离临时建筑物和广告牌
  • 台风期间禁止外出

4. 泥石流避险

  • 立即向两侧高处逃离
  • 不要顺着泥石流方向跑
  • 不要在谷底停留
  • 爬到树上暂时躲避（不得已时）

5. 雷电避险

  • 不要在大树下躲避
  • 不要在山顶或高地
  • 远离铁塔、电线杆
  • 在室内时关闭门窗

6. 高温中暑避险

  • 立即转移到阴凉处
  • 解开衣领散热
  • 用湿毛巾冷敷头部
  • 补充淡盐水
  • 严重者立即送医
★ 自然灾害避险共同原则：
  1. 保持冷静，听从指挥
  2. 保护头部最重要
  3. 迅速转移到安全地带
  4. 及时清点人数
  5. 立即报告旅行社

四、游客投诉的处理

★1. 处理投诉的基本原则

  • 真诚倾听
  • 耐心解释
  • 维护游客尊严
  • 不推卸责任
  • 及时处理

2. 处理投诉的程序

  1. 认真倾听投诉内容
  2. 对游客表示同情和理解
  3. 对游客的投诉表示感谢
  4. 调查核实情况
  5. 提出解决方案
  6. 将处理结果反馈给游客
  7. 做好记录存档
