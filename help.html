🎒 游客服务

如何注册游导旅游账号？

注册游导旅游账号非常简单：

1. 点击页面右上角的"注册"按钮
2. 选择身份类型（游客/导游）
3. 填写手机号并获取验证码
4. 设置登录密码
5. 完成基础信息填写
6. 验证邮箱（可选但推荐）
7. 注册成功，开始使用平台服务

如何预约导游？

预约导游的步骤如下：

1. 浏览导游：在"找导游"页面浏览导游列表，可按目的地、语言、评分筛选
2. 查看详情：点击导游头像查看详细信息，包括简介、服务范围、用户评价
3. 发起预约：确定导游后，点击"立即预约"，填写旅行日期、人数、行程需求
4. 等待确认：导游会在24小时内确认预约
5. 支付定金：确认后支付服务定金（通常是总费用的30%）
6. 出行前沟通：导游会提前与您联系确认行程细节
7. 完成服务：旅行结束后支付尾款，并可对导游进行评价

可以取消或更改预约吗？

可以，但需要注意以下规则：

• 免费取消：出行日期前7天以上可免费取消
• 部分退款：出行日期前3-7天取消，退还50%定金
• 不可退款：出行日期前3天内取消，不退还定金
• 更改日期：如需更改日期，请提前3天联系导游协商
• 特殊情况：如遇不可抗力因素（天气、疾病等），可与导游协商延期或退款

导游服务质量有保障吗？

我们非常重视服务质量保障：

1. 严格审核：所有入驻导游都需通过资质审核，包括身份认证、专业资质验证
2. 评分系统：用户可在服务结束后对导游进行评分和评价
3. 投诉机制：如遇服务问题，可在平台发起投诉，我们会及时处理
4. 先行赔付：经核实确属导游责任的，平台可提供先行赔付
5. 服务质量追踪：平台会定期评估导游服务质量，不合格者将被下架

如何联系导游？

您可以通过以下方式联系导游：

• 平台消息：登录后通过"我的订单"中的聊天功能与导游沟通
• 预约确认后：预约确认后可以看到导游的联系方式
• 紧急情况：如有紧急问题，可联系平台客服协助

儿童需要单独预约导游吗？

这取决于具体情况：

• 亲子游：如果您是带孩子旅行，部分导游专门提供亲子游服务，可一起预约
• 大龄儿童：10岁以上儿童通常可以和成人一起计入人数
• 幼童：10岁以下儿童可能需要导游额外照顾，建议提前与导游沟通
• 费用说明：儿童费用视具体服务内容而定，请在预约时说明

🎤 导游入驻

如何申请成为平台导游？

申请入驻流程：

1. 注册账号：在首页选择"我是导游"，完成基础注册
2. 选择身份：选择导游身份并填写详细信息
3. 上传资质：提交身份证、导游证、语言能力证明等
4. 完善资料：填写个人简介、可服务地区、擅长领域
5. 等待审核：平台会在3个工作日内完成审核
6. 通过上线：审核通过后，您的导游主页将正式上线

→ 立即申请入驻

入驻需要哪些资质？

平台对导游资质有严格要求：

• 基本要求：年满18周岁，身体健康
• 身份认证：有效的身份证件
• 专业资质（满足其一即可）：
   - 持有国家旅游局颁发的导游资格证
   - 持有境外导游/领队相关资质
   - 具有丰富的当地生活/旅游经验（需审核）
• 语言能力：如服务外语游客，需提供相应语言能力证明
• 其他：良好的沟通能力、服务意识

入驻审核需要多长时间？

审核时间说明：

• 常规审核：提交完整材料后3个工作日内完成
• 加急审核：如急需上线，可联系客服申请加急（1个工作日）
• 补充材料：如需补充材料，审核时间会相应延长
• 审核结果：审核结果会通过站内消息和短信通知

导游可以同时服务多个游客吗？

可以的，但需要注意：

• 时间不冲突：确保不同游客的服务时间不重叠
• 如实告知：如需同时服务多个游客，应提前告知各方
• 服务质量：必须保证对每个游客的服务质量
• 价格透明：如有额外服务费用，需提前说明

如何提升导游排名和曝光？

提升排名的技巧：

1. 完善资料：头像、简介、服务描述越详细越好
2. 获得好评：服务质量越好，好评越多，排名越高
3. 及时回复：快速响应游客咨询和消息
4. 保持在线：经常上线，更新可服务时间
5. 服务特色：提供特色服务，如定制行程、特色体验
6. 参与活动：积极参与平台举办的各类活动

导游可以拒绝服务吗？

在以下情况下，导游可以礼貌拒绝：

• 时间冲突：已有其他预约时间冲突
• 能力范围：超出自己擅长的服务领域
• 不合理要求：游客提出违法或有违公序良俗的要求
• 特殊情况：如遇不可抗力因素

⚠️ 注意：请在拒绝前与游客充分沟通，尽量提供替代建议。

💰 费用与支付

游导旅游平台收费吗？

平台对游客免费，导游需支付服务费：

• 游客免费：游客浏览导游、预约服务均免费
• 导游服务费：导游每单需支付一定比例的平台服务费（约10-15%）
• 提现手续费：导游提现时可能产生少量银行手续费
• 推广服务：导游可自愿购买推广服务提升曝光

导游服务费用如何计算？

导游服务费用由以下因素决定：

• 服务时长：按天、半日或小时计费
• 导游资质：经验丰富、评分高的导游可能收费更高
• 目的地：热门旅游目的地价格相对较高
• 服务内容：定制行程、包车服务等可能有额外费用
• 淡旺季：旅游旺季价格可能上调

💡 建议：预约前与导游详细沟通服务内容和费用明细。

支持哪些支付方式？

我们支持多种支付方式：

• 微信支付：推荐使用，支付便捷
• 支付宝：支持花呗、信用卡
• 银行卡：支持主流银行的储蓄卡和信用卡
• 平台余额：可充值到平台余额，下次直接使用

如何申请退款？

退款规则：

1. 取消预约退款：根据取消时间退还相应比例（详见预约取消规则）
2. 服务未开始：如导游原因导致服务未开始，可申请全额退款
3. 服务质量问题：如对服务不满意，可先与导游沟通，协商不成可申请平台介入
4. 退款流程：进入"我的订单"→选择订单→点击"申请退款"→填写原因→等待审核
5. 退款时间：审核通过后3-7个工作日内退回原支付渠道

导游如何提现收入？

导游提现说明：

• 提现条件：账户余额满100元即可提现
• 提现方式：支持绑定银行卡、微信零钱
• 到账时间：申请后1-3个工作日到账
• 手续费：每月前3次提现免费，之后收取1%手续费（最低1元）
• 提现路径：导游工作台 → 财务管理 → 申请提现

🔒 隐私与安全

我的个人信息安全吗？

我们非常重视用户隐私保护：

• 数据加密：所有敏感数据采用银行级加密传输和存储
• 隐私政策：严格遵守隐私政策，不会向第三方泄露用户信息
• 权限控制：只有必要的工作人员在必要范围内可访问用户数据
• 定期审计：定期进行安全审计和漏洞修复
• 合规认证：符合国家相关法律法规要求

→ 查看完整隐私政策

我的联系方式会被导游看到吗？

关于联系方式的说明：

• 预约前：游客和导游在预约确认前互相看不到对方完整联系方式
• 预约后：预约确认后，可看到对方的部分联系方式以便沟通行程
• 隐私保护：真实手机号会被隐藏，中间4位用星号代替
• 平台沟通：建议优先使用平台内置聊天功能进行沟通

如何设置账户安全？

保护账户安全的建议：

1. 强密码：设置包含字母、数字、符号的复杂密码
2. 不重复：不要在多个网站使用相同密码
3. 绑定手机：务必绑定手机号，用于找回密码和验证
4. 开启验证：重要操作时开启短信验证码
5. 谨慎点击：不要点击陌生链接，防止钓鱼网站
6. 定期更换：建议定期更换登录密码

遇到诈骗或虚假信息怎么办？

如遇可疑情况，请立即：

1. 不要转账：切勿向陌生人转账，务必通过平台支付
2. 保留证据：截图保存聊天记录和交易凭证
3. 举报用户：通过平台举报功能举报可疑用户
4. 联系客服：拨打客服热线或通过在线客服求助
5. 报警处理：如已造成损失，请及时向公安机关报案

→ 立即联系客服

👤 账户问题

忘记密码怎么办？

找回密码步骤：

1. 点击登录页的"忘记密码"链接
2. 输入注册时使用的手机号或邮箱
3. 获取验证码
4. 设置新密码
5. 使用新密码登录

如何修改个人信息？

修改个人信息：

• 登录账号：进入"个人中心"
• 头像昵称：点击头像进入个人资料页面修改
• 手机号：进入"账号设置"→"手机绑定"更换
• 邮箱：进入"账号设置"→"邮箱绑定"更换
• 密码：进入"账号设置"→"修改密码"更新

账号被封禁怎么办？

如账号被封禁：

1. 了解原因：查看封禁通知了解封禁原因
2. 申诉入口：通过封禁通知中的链接提交申诉
3. 联系客服：如未找到申诉入口，可联系客服
4. 提供证明：如认为封禁有误，准备相关证明材料
5. 等待处理：申诉结果会在1-3个工作日内通知

如何注销账号？

注销账号须知：

• 注销条件：账号内无进行中的订单、无未提现余额
• 注销路径：个人中心 → 账号设置 → 注销账号
• 生效时间：申请注销后7天内生效
• 数据清除：注销后个人数据将无法恢复，请谨慎操作

✈️ 旅行相关

护照过期了还能出国吗？

不能。大多数国家要求护照有效期在6个月以上（从入境日期算起），部分国家要求3个月以上。建议护照有效期不足1年时及时更换。

签证被拒了怎么办？

签证被拒后的处理方法：

1. 了解拒签原因
2. 准备更充分的材料重新申请
3. 申诉（如有充分理由）
4. 考虑申请其他类型签证
5. 等待适当时间后再申请

→ 查看签证攻略

机票什么时候订最便宜？

机票预订建议：

• 国内航班：提前1-2个月
• 国际航班：提前2-3个月
• 周二、周四出发通常较便宜
• 关注航空公司大促（双11、黑五、周年庆）
• 使用比价网站设置价格提醒

护照丢失怎么办？

护照丢失紧急处理：

1. 立即报警，获取报警记录
2. 联系中国驻当地使领馆
3. 申请旅行证（所需材料：报警记录、护照复印件、照片）
4. 旅行证有效期有限，需尽快回国
5. 回国后到出入境管理局补办护照

→ 查看应急资源

需要买旅行保险吗？

强烈建议购买旅行保险！可覆盖：

• 医疗费用（海外就医可能很贵）
• 航班延误/取消
• 行李丢失
• 意外伤害
• 紧急救援
注意：申根国家要求必须购买保险。

手机在国外怎么上网？

三种上网方式：

1. 国际漫游：向运营商开通，费用较高
2. 当地SIM卡：到达后购买，价格实惠
3. 移动WiFi：可多人共享，适合团队

💡 建议提前在网上购买当地流量包最方便。
