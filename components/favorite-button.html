使用说明

引入组件

<script src="/components/favorite-button.js"></script>

基础用法

<favorite-button item-id="guide-001"></favorite-button>

完整属性

<favorite-button
    item-id="guide-001"
    size="md"
    style="default"
    animation="pop"
    color="red"
    initial-count="123"
    show-count="true"
></favorite-button>

JavaScript API

// 获取组件实例
const btn = document.querySelector('favorite-button');

// 监听收藏状态变化
btn.addEventListener('favorite-change', (e) => {
    console.log('收藏状态:', e.detail.isFavorited);
    console.log('收藏数量:', e.detail.count);
});

// 手动触发收藏/取消收藏
btn.toggle();

// 检查是否已收藏
console.log(btn.isFavorited);

// 获取/设置收藏数量
console.log(btn.count);
btn.count = 100;

属性说明

属性 可选值 说明
item-id string 收藏项唯一ID（必填）
size sm / md / lg 按钮尺寸，默认 md
style default / solid / outline / label / count / float / overlay 按钮样式，默认 default
animation pop / beat / shake / ripple 动画效果，默认 pop
color red / pink / purple / blue 图标颜色，默认 red
initial-count number 初始收藏数量
show-count boolean 是否显示数量，默认 false
