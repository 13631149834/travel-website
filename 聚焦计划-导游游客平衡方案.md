# 游导伴旅 - 聚焦计划

## 核心定位

> **不做OTA（在线旅行社），做导游与游客的"连接器"**
> 
> 帮游客做决策，帮导游提效率，不碰交易环节

---

## 一、为什么这样定位？

| 对比 | OTA模式（携程/飞猪） | 我们的模式 |
|------|---------------------|-----------|
| 核心业务 | 卖机票、酒店、旅行团 | 提供工具和内容 |
| 盈利方式 | 交易抽佣 | 增值服务/广告 |
| 导游角色 | 供应商/执行者 | 独立服务者 |
| 和旅行社关系 | 竞争 | 不触碰 |

**结论：避开巨头竞争，找到差异化定位**

---

## 二、两端价值主张

### 游客端：旅行决策助手

**一句话定位：** 帮游客规划旅程，找到靠谱导游

| 功能 | 价值 | 使用场景 |
|------|------|---------|
| 智能行程规划 | 输入目的地天数，AI生成行程 | 出行前规划 |
| 签证攻略中心 | 签证材料、流程、常见问题 | 签证办理 |
| 避坑指南 | 46国真实避坑经验 | 出行准备 |
| 导游搜索 | 按目的地/语种/专长找导游 | 想找靠谱导游 |
| 导游主页 | 看资质、评价、案例 | 判断导游靠不靠谱 |

**核心体验：**
```
游客进入网站
    ↓
输入：想去日本，5天，2大1小
    ↓
AI生成行程建议 + 推荐相关导游
    ↓
游客联系导游（微信/电话）
    ↓
成交在平台外完成
```

---

### 导游端：效率工具箱

**一句话定位：** 让导游看起来更专业、工作更高效

| 功能 | 价值 | 使用场景 |
|------|------|---------|
| 行程单生成器 | AI生成专业行程，一键导出 | 接团前准备 |
| 导游个人主页 | 展示资质、案例、评价 | 客户了解自己 |
| 客户管理 | 记录客人信息、提醒跟进 | 团期管理 |
| 知识速查 | 快速查景点、应急电话 | 带团中 |

**核心体验：**
```
导游进入网站
    ↓
创建个人主页（上传证书、填写擅长）
    ↓
使用行程生成器，快速出方案
    ↓
把主页链接发给客户
    ↓
客户看到专业展示，信任度↑
```

## 三、平衡点：连接但不交易

### 关键设计

```
┌─────────────────────────────────────────────────────────┐
│                      游导伴旅平台                        │
├──────────────────────┬──────────────────────────────────┤
│       游客端          │            导游端               │
├──────────────────────┼──────────────────────────────────┤
│ • AI行程规划          │ • 行程单生成器                  │
│ • 签证/避坑攻略       │ • 个人主页                      │
│ • 搜索导游            │ • 客户管理                      │
│ • 查看导游主页        │ • 知识速查                      │
└──────────────────────┴──────────────────────────────────┘
                    ↓
         【连接层：但不碰交易】
                    ↓
┌─────────────────────────────────────────────────────────┐
│  导游主页展示：微信二维码 / 电话 / 私信入口             │
│  平台不参与：报价、收款、合同、售后                     │
└─────────────────────────────────────────────────────────┘
```

### 这样做的好处

| 方面 | 好处 |
|------|------|
| **法律风险** | 平台不涉及旅游业务资质问题 |
| **旅行社关系** | 不抢客户，只是帮导游展示 |
| **导游信任** | 导游不担心平台抽成 |
| **运营成本** | 无需处理支付、退款、投诉 |

## 六、和旅行社的关系处理

### 核心原则

> **我们帮导游展示专业，不帮导游抢客户**

### 具体做法

| 情况 | 处理方式 |
|------|---------|
| 导游是旅行社员工 | 主页标注"XX旅行社导游"，游客联系旅行社 |
| 导游是自由执业 | 主页展示联系方式，游客直接联系导游 |
| 旅行社入驻 | 可创建旅行社主页，展示旗下导游 |

### 话术

对旅行社说：
> "我们做的是导游工具，帮您的导游工作效率更高、展示更专业。游客看到的是'XX旅行社的张导'，品牌还是您的。"

对导游说：
> "我们帮您建立个人品牌，积累客户评价。客户来源还是靠旅行社分配，但您的主页让客户更信任您。"

## 七、总结

### 一句话定位

| 端 | 定位 |
|---|------|
| **游客端** | 旅行规划助手，帮你做决策、找导游 |
| **导游端** | 效率工具箱，让你更专业、更高效 |
| **平台** | 连接器，不碰交易 |

### 成功关键

1. **工具好用** - 行程生成器要真的方便
2. **导游愿意用** - 省时间、显得专业
3. **游客愿意看** - 内容有用、导游靠谱
4. **不触碰交易** - 避开法律风险和行业冲突

### 第一步做什么

**先做行程生成器 + 导游主页模板**

这是两端都能用的核心功能，做好了就有第一批用户。
