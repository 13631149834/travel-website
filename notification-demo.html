📖 功能说明

🔔 通知类型

  • 系统通知 - 网站更新、维护
  • 活动通知 - 促销、优惠活动
  • 订单通知 - 订单状态变更
  • 预约提醒 - 行程、导游预约

⚙️ 设置选项

  • 浏览器通知 - 桌面弹窗
  • 站内消息 - Toast提示
  • 声音提醒 - 提示音
  • 角标提醒 - 未读数量

💾 数据存储

  • 本地存储 - localStorage
  • 最多保存100条通知
  • 自动清理过期数据
  • 设置跨页面同步

💻 API 调用示例

// 获取通知实例
const notif = notificationSystem;

// 系统通知
notif.systemNotify('版本更新', '网站已更新至V2.0版本');

// 活动通知
notif.activityNotify('限时优惠', '新用户首单立减50元！');

// 订单通知
notif.orderNotify('订单已确认', '您的日本行程订单已确认');

// 预约提醒
notif.reminderNotify('明日行程', '明天9:00与导游张明的会面');

// 获取未读数
const count = notif.getUnreadCount();

// 标记已读
notif.markAsRead('notification_id');

// 标记全部已读
notif.markAllAsRead();

// 清空通知
notif.clearAll();
