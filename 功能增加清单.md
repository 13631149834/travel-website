## 一、登录/注册系统升级 🔐（优先）

### 1.1 游客与导游分离

**现状：** 登录页面混在一起，不够清晰

**升级方案：**

| 入口 | 设计 |
|------|------|
| 首页登录按钮 | 改为下拉菜单：「游客登录」「导游登录」 |
| 登录页面 | 顶部Tab切换：「我是游客」「我是导游」 |
| 注册页面 | 分离为两个独立页面 |

### 1.2 导游注册增强

**必须填写的信息：**

| 字段 | 类型 | 说明 |
|------|------|------|
| 真实姓名 | 文本 | 用于审核 |
| 导游证号 | 文本 | 必填，格式校验 |
| 导游证照片 | 图片上传 | 必填 |
| 从业年限 | 下拉选择 | 1-3年/3-5年/5-10年/10年以上 |
| 带团数量 | 下拉选择 | 10团以下/10-50团/50-100团/100团以上 |
| 擅长领域 | 多选 | 欧洲/日本/东南亚/澳新/中东/国内 |
| 服务语言 | 多选 | 普通话/英语/日语/韩语/粤语等 |
| 所属旅行社 | 文本 | 可选 |
| 个人简介 | 文本域 | 100字以内 |

### 1.3 导游身份验证

**流程：**
```
提交注册信息
    ↓
系统标记为"待审核"
    ↓
管理员后台审核导游证
    ↓
审核通过 → 可使用导游功能
审核拒绝 → 提示原因，可重新提交
```
