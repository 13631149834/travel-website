# 第三、四阶段开发计划（本地版）

## 开发目标
在不需要联网/云数据库的情况下，完成以下功能的本地版本开发。

---

## 第三阶段功能

### 1. 导游个人主页 (profile.html)

#### 功能清单
- [ ] 个人信息编辑
  - 姓名、昵称
  - 头像上传（Base64存储）
  - 个人简介
  - 从业年限
  - 服务语言选择

- [ ] 资质证书展示
  - 导游证号
  - 导游等级（初级/中级/高级/特级）
  - 证书照片上传
  - 擅长领域标签

- [ ] 服务范围
  - 擅长线路列表（可添加/编辑/删除）
  - 服务城市选择
  - 服务类型（跟团/定制/私家团）

- [ ] 联系方式
  - 手机号
  - 微信号
  - 邮箱
  - 在线咨询按钮

- [ ] 分享功能
  - 生成个人主页二维码
  - 复制链接分享
  - 分享到社交媒体

- [ ] 页面预览
  - 模拟客户视角查看
  - 打印简历版本

#### 技术实现
- localStorage存储所有数据
- Base64编码存储图片
- qrcode.js生成二维码
- 响应式设计

---

### 2. 客人评价系统 - 本地版 (reviews.html)

#### 功能清单
- [ ] 评价入口
  - 展示二维码（游客扫码评价）
  - 本地评价表单入口

- [ ] 评价表单
  - 导游选择
  - 服务态度评分（1-5星）
  - 专业程度评分（1-5星）
  - 行程安排评分（1-5星）
  - 综合评分（自动计算）
  - 文字评价
  - 游客姓名（可选）
  - 评价日期

- [ ] 评价展示
  - 好评墙（高分评价展示）
  - 评价列表（全部评价）
  - 评分统计（平均分、评价数）

- [ ] 评价管理
  - 查看所有评价
  - 回复评价
  - 删除不当评价
  - 标记精选评价

- [ ] 数据统计
  - 评分趋势图
  - 服务态度分布
  - 专业程度分布
  - 行程安排分布

- [ ] 数据导出
  - 导出CSV
  - 导出PDF（好评墙）

#### 技术实现
- localStorage存储评价
- Chart.js绘制统计图表
- 二维码生成（qrcode.js）
- 响应式设计

## 第四阶段功能

### 3. 学习成长体系 (learning.html)

#### 功能清单
- [ ] 导游考试题库
  - 科目分类（政策法规/导游业务/全国导游基础知识/地方导游基础知识）
  - 题型分类（单选/多选/判断/简答）
  - 题目数量：至少100题
  - 答案解析

- [ ] 练习模式
  - 随机练习
  - 分类练习
  - 错题重练
  - 收藏题目

- [ ] 模拟考试
  - 计时功能
  - 自动评分
  - 成绩记录
  - 历史考试记录

- [ ] 学习进度
  - 已学题目统计
  - 正确率统计
  - 学习时长记录
  - 学习曲线图

- [ ] 错题本
  - 自动收录错题
  - 错题分类
  - 错题复习

- [ ] 成就系统
  - 学习徽章（如：学习达人、答题高手、坚持打卡）
  - 等级系统（青铜/白银/黄金/钻石）
  - 积分系统

- [ ] 知识库
  - 导游必备知识
  - 政策法规解读
  - 行业动态
  - 实用技巧

#### 技术实现
- localStorage存储学习数据
- 题库数据（JSON格式）
- Chart.js绘制学习曲线
- 成就系统（徽章图标）

## 设计要求

### 风格统一
- 与现有网站保持一致
- 蓝紫渐变配色
- 卡片式布局
- 圆角设计
- 流畅动画

### 响应式设计
- 移动端优先
- 触屏友好
- 大按钮设计

### 用户体验
- 操作简单直观
- 数据自动保存
- 清晰的操作反馈
- 数据导出功能

---

## 开发顺序

1. **导游个人主页**（2-3小时）
   - 最重要，展示个人品牌
   - 先做基础信息编辑
   - 再做二维码分享

2. **学习成长体系**（3-4小时）
   - 题库数据准备
   - 练习功能
   - 学习统计
   - 成就系统

3. **客人评价系统**（3-4小时）
   - 评价表单
   - 评价展示
   - 数据统计

---

## 注意事项

1. **数据存储**
   - 使用localStorage
   - 提供导出功能
   - 定期提醒备份

2. **移动端适配**
   - 所有功能必须在手机上可用
   - 字体大小适中
   - 按钮足够大

3. **性能优化**
   - 题库数据分批加载
   - 图片压缩存储
   - 避免内存泄漏

4. **用户引导**
   - 首次使用引导
   - 功能说明提示
   - 帮助文档链接
