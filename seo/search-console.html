Google Search Console

全球最大搜索引擎 - 必备工具

1 添加网站资源

访问 Google Search Console 并添加您的网站。

两种添加方式：

  • 网域资源：输入 youdao-travel.com（推荐，自动包含所有子域名）
  • 网址前缀：输入完整URL如 https://youdao-travel.com/

2 验证网站所有权

Google提供多种验证方式，选择最适合您的方式：

✅ HTML文件上传（推荐）

  1. 下载Google提供的HTML验证文件
  2. 将文件上传到网站根目录
  3. 确保文件可通过 https://youdao-travel.com/googlexxxx.html 访问
  4. 点击"验证"按钮
将 googlexxxx.html 上传到 ./travel-website/ 目录

✅ HTML标签

将以下meta标签添加到首页 <head> 中：

<meta name="google-site-verification" content="您的验证码" />

✅ Google Analytics

如果您已安装GA，只需确认有编辑权限即可自动验证

✅ Google Tag Manager

如果您已安装GTM容器，验证容器片段即可

3 提交网站地图

网站地图帮助Google更好地发现和抓取您的页面：

  1. 进入左侧菜单「网站地图」
  2. 在"添加网站地图"输入框中输入 sitemap.xml
  3. 点击提交按钮
  4. 等待状态变为"成功"（通常需要几分钟到24小时）

您的网站地图地址： https://youdao-travel.com/sitemap.xml

百度搜索资源平台

中国市场必备 - 最重要的中文搜索引擎

1 添加网站

访问 百度搜索资源平台 并登录百度账号。

提示：建议使用百度账号登录，与百度其他产品（如百度统计）可共用数据。

2 选择验证方式

百度提供多种验证方式，推荐HTML标签验证：

HTML标签验证（推荐）

将以下meta标签添加到首页 <head> 中：

<meta name="baidu-site-verification" content="您的验证码" />

文件验证

下载验证文件，上传到网站根目录

CNAME验证

添加DNS解析记录

✅ 已验证：网站已配置百度站点验证

<meta name="baidu-site-verification" content="baidu_verify_code" />

3 提交收录

百度提供多种提交方式：

✓

  • • 每日配额：1000条

📈 数据监控指南

定期监控搜索数据是SEO工作的重要部分。以下是关键指标和监控方法：

📊 核心监控指标

-
展示次数 (Impressions)
用户在搜索结果中看到您网站的次数
-
点击次数 (Clicks)
用户点击进入您网站的次数
-
点击率 (CTR)
点击次数/展示次数
-
平均排名
关键词的平均搜索位置

📅 建议监控频率

每周 检查排名变化、排查抓取错误、查看新增外链
每月 分析整体趋势、对比上月数据、制定优化计划
每季度 全面SEO审计、竞争对手分析、策略调整

⚠️ 设置异常告警

🔴

排名大幅下降

如果核心关键词排名下降超过5位，需要立即排查原因

🟡

抓取错误增加

发现大量404或其他错误时需要处理

🔵

收录量下降

如果索引页面数量明显减少，需要检查原因

🚀 快速收录提交方法

对于重要的新页面或更新内容，建议主动提交以加速收录。

🔵 Google收录提交

  1. 1

    打开URL检查工具

    在Google Search Console中进入「URL检查」工具

  2. 2

    输入页面URL

    输入要提交的页面完整URL

  3. 3

    请求编制索引

    如果页面未收录，点击"请求编制索引"按钮

🔴 百度收录提交

主动推送（最快）

通过API实时推送URL列表：

curl -H 'Content-Type:application/json' -X POST -d '{"urls":["https://youdao-travel.com/new-page.html"]}' 'https://ziyuan.baidu.com/urls?site=youdao-travel.com&token=您的TOKEN'

自动推送（最省心）

在页面footer中添加以下JS代码：

<script>(function(){var bp=document.createElement('script');bp.src='//push.zhanzhang.baidu.com/push.js';var s=document.getElementsByTagName("script")[0];s.parentNode.insertBefore(bp,s);})();</script>

💡 收录提交最佳实践

  • • 优先提交高质量、原创、有价值的内容页面
  • • 确保页面可正常访问，无404或其他错误
  • • 已收录页面无需重复提交，只需提交新内容或重大更新
  • • 避免批量提交大量重复或低质量页面
  • • 保持稳定的更新频率比一次性大量提交更有效
