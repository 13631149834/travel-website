📝 Meta标签配置

必需 标题标签 (Title Tag)

🎯 最佳长度：50-60个字符（包含空格）

确保核心关键词靠前，使用管道符或短横线分隔

<title>游导旅游 - 专业导游预约平台 | 严选认证导游</title>

✅ 正确示例

游导旅游 - 专业导游预约平台

长度：16字符 ✓

❌ 错误示例

欢迎来到我们的网站，这里提供最专业的旅游服务，包括导游预约、行程规划...

长度超过60字符，被搜索引擎截断

必需 描述标签 (Meta Description)

🎯 最佳长度：150-160个字符

包含核心关键词和行动号召，撰写吸引点击的描述

<meta name="description" content="游导旅游是专业的导游预约平台，严选认证导游，提供一对一专属服务。支持在线预约、实时沟通，让您的旅行更简单。立即注册，享受首单优惠！">

可选 关键词标签 (Keywords)

⚠️ 虽被Google放弃，但百度仍参考，建议合理使用

<meta name="keywords" content="导游预约,旅行平台,导游服务,旅游攻略,行程规划,景点讲解">

必需 视口设置 (Viewport) 

<meta name="viewport" content="width=device-width, initial-scale=1.0">

必需 爬虫指令 (Robots) 

<meta name="robots" content="index, follow">
<meta name="googlebot" content="index, follow">
index, follow
允许抓取和跟踪链接
noindex, follow
禁止抓取但跟踪链接
noindex, nofollow
完全禁止

必需 规范链接 (Canonical)

避免重复内容问题，指定页面的规范版本

<link rel="canonical" href="https://youdao-travel.com/destinations.html">

必需 语言声明 

<html lang="zh-CN">

📱 Open Graph社交分享配置

Open Graph协议由Facebook创建，用于控制在社交媒体分享时显示的内容。配置后，分享链接将显示精美的预览卡片。

<!-- 基础Open Graph标签 -->
<meta property="og:type" content="website">
<meta property="og:title" content="游导旅游 - 专业导游预约平台">
<meta property="og:description" content="严选认证导游，提供一对一专属服务，支持在线预约，让旅行更简单">
<meta property="og:image" content="https://youdao-travel.com/images/og-image.jpg">
<meta property="og:url" content="https://youdao-travel.com/">
<meta property="og:site_name" content="游导旅游">

<!-- 图片要求 -->
<meta property="og:image:width" content="1200">
<meta property="og:image:height" content="630">
<meta property="og:image:alt" content="游导旅游平台界面">

✅ 推荐图片规格

  • • 最小尺寸：1200×630像素
  • • 推荐比例：1.91:1
  • • 文件大小：小于1MB
  • • 格式：JPG、PNG、WebP

📋 og:type 类型参考

  • • website - 网站首页
  • • article - 文章/博客
  • • profile - 人物/导游
  • • product - 商品/服务

🏷️ 结构化数据配置 (Schema.org)

结构化数据帮助搜索引擎理解页面内容，可提升搜索结果展示效果，如富媒体摘要、星级评分、知识卡片等。

🏢 Organization (组织信息)

放置在首页，用于定义网站所属组织

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "游导旅游",
  "url": "https://youdao-travel.com",
  "logo": "https://youdao-travel.com/images/logo.png",
  "description": "专业导游预约平台，严选认证导游",
  "foundingDate": "2020",
  "contactPoint": {
    "@type": "ContactPoint",
    "telephone": "+86-13631149834",
    "contactType": "customer service",
    "availableLanguage": ["Chinese", "English"]
  },
  "sameAs": [
    "https://weibo.com/youdaotravel",
    "https://www.xiaohongshu.com/youdaotravel",
    "https://www.douyin.com/youdaotravel"
  ]
}
</script>

🌐 WebSite (网站信息)

包含搜索框结构化数据，便于搜索引擎显示站内搜索

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "name": "游导旅游",
  "url": "https://youdao-travel.com",
  "potentialAction": {
    "@type": "SearchAction",
    "target": "https://youdao-travel.com/search.html?q={search_term_string}",
    "query-input": "required name=search_term_string"
  }
}
</script>
