🗺️ 什么是网站地图？

网站地图是一个XML文件，列出了网站上所有重要页面的URL，帮助搜索引擎蜘蛛更高效地抓取和索引您的网站内容。

🔍

提升抓取效率

帮助搜索引擎发现新页面和更新内容

📊

结构化呈现

提供页面层级关系和重要性信息

⚡

加速索引

缩短页面被搜索收录的时间

📄 XML网站地图创建

📋 标准网站地图模板

<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://youdao-travel.com/</loc>
    <lastmod>2024-01-20</lastmod>
    <changefreq>daily</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://youdao-travel.com/destinations.html</loc>
    <lastmod>2024-01-20</lastmod>
    <changefreq>weekly</changefreq>
    <priority>0.9</priority>
  </url>
  <url>
    <loc>https://youdao-travel.com/guides.html</loc>
    <lastmod>2024-01-20</lastmod>
    <changefreq>daily</changefreq>
    <priority>0.9</priority>
  </url>
</urlset>

📝 标签说明

  • <loc> - 页面URL地址（必需，绝对路径，必须以http/https开头）
  • <lastmod> - 最后修改时间（可选，建议使用ISO 8601格式）
  • <changefreq> - 更新频率（可选，用于提示搜索引擎）
  • <priority> - 优先级（可选，0.0-1.0，相对值）

🖼️ 图片站点地图

如果您有大量图片内容，可以通过图片站点地图帮助搜索引擎发现和索引这些图片，提升图片搜索流量。

<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9"
        xmlns:image="http://www.google.com/schemas/sitemap-image/1.1">
  <url>
    <loc>https://youdao-travel.com/destinations/yunnan.html</loc>
    <image:image>
      <image:loc>https://youdao-travel.com/images/yunnan-cover.jpg</image:loc>
      <image:title>云南旅游攻略 - 丽江大理香格里拉</image:title>
      <image:caption>云南最美风景 - 玉龙雪山</image:caption>
    </image:image>
    <image:image>
      <image:loc>https://youdao-travel.com/images/yunnan-2.jpg</image:loc>
      <image:title>云南旅行 - 大理古城</image:title>
    </image:image>
  </url>
  <url>
    <loc>https://youdao-travel.com/guides.html</loc>
    <image:image>
      <image:loc>https://youdao-travel.com/images/guide-team.jpg</image:loc>
      <image:title>游导旅游 - 认证导游团队</image:title>
    </image:image>
  </url>
</urlset>

📋 图片标签说明

  • image:loc - 图片URL（必需）
  • image:title - 图片标题（可选）
  • image:caption - 图片说明（可选）
  • image:geo_location - 地理位置（可选）
  • image:license - 许可证（可选）

💡 最佳实践

  • • 为每张图片添加描述性alt文本
  • • 确保图片URL可公开访问
  • • 使用高分辨率图片
  • • 避免重复图片URL

📚 网站地图索引 (Sitemap Index)

当网站规模较大时，建议将网站地图拆分为多个文件，然后创建索引文件。每个网站地图文件最多包含50,000个URL。

📄 sitemap_index.xml

<?xml version="1.0" encoding="UTF-8"?>
<sitemapindex xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <sitemap>
    <loc>https://youdao-travel.com/sitemap.xml</loc>
    <lastmod>2024-01-20</lastmod>
  </sitemap>
  <sitemap>
    <loc>https://youdao-travel.com/sitemap-articles.xml</loc>
    <lastmod>2024-01-20</lastmod>
  </sitemap>
  <sitemap>
    <loc>https://youdao-travel.com/sitemap-images.xml</loc>
    <lastmod>2024-01-20</lastmod>
  </sitemap>
  <sitemap>
    <loc>https://youdao-travel.com/sitemap-guides.xml</loc>
    <lastmod>2024-01-20</lastmod>
  </sitemap>
</sitemapindex>

⚠️ 重要限制

  • • 单个网站地图文件最多 50,000 个URL
  • • 单个网站地图文件最大 50MB（未压缩）
  • • 如果需要超过限制，创建多个网站地图并使用索引
  • • 所有URL必须在同一个主机上

🤖 robots.txt 配置

在网站根目录创建 robots.txt 文件，告诉搜索引擎哪些页面可以抓取，以及网站地图的位置。

📄 robots.txt

# robots.txt for youdao-travel.com

# 允许所有搜索引擎访问
User-agent: *

# 允许访问的路径
Allow: /

# 禁止访问的路径
Disallow: /admin/
Disallow: /api/
Disallow: /private/
Disallow: /checkout/
Disallow: /*.json$
Disallow: /search?q=

# 网站地图位置
Sitemap: https://youdao-travel.com/sitemap.xml
Sitemap: https://youdao-travel.com/sitemap-images.xml

# 其他搜索引擎特定规则
User-agent: Googlebot
Allow: /

User-agent: Baiduspider
Allow: /

🚀 提交到搜索引擎

🔵

Google Search Console

全球最大搜索引擎

  1. 1. 访问 Google Search Console
  2. 2. 添加并验证您的网站
  3. 3. 进入「网站地图」栏目
  4. 4. 输入 sitemap.xml 并提交
https://youdao-travel.com/sitemap.xml
🔴

百度搜索资源平台

中国市场首选

  1. 1. 访问 百度搜索资源平台
  2. 2. 完成站点验证（支持HTML标签验证）
  3. 3. 进入「数据提交」→「网站地图」
  4. 4. 提交网站地图文件
支持自动推送和主动推送
🟣

Bing Webmaster

微软必应搜索

  1. 1. 访问 Bing Webmaster Tools
  2. 2. 添加并验证网站
  3. 3. 进入「配置我的网站」→「网站地图」
  4. 4. 提交网站地图URL
🟢

360搜索站长平台

中国市场补充

  1. 1. 访问 360站长平台
  2. 2. 验证网站所有权
  3. 3. 在「网站管理」中添加网站
  4. 4. 提交网站地图
