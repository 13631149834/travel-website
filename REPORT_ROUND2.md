## 第1轮：全站视觉一致性深度审查

### 发现了什么
1. **glass-card 圆角不统一**：index.html=14px，其他页面=20px
2. **key-point 颜色不一致**：部分页面用 #1A237E，部分用 #4C8BF5
3. **阴影风格多样**：存在 rgba(0,0,0,0.04)、rgba(0,0,0,0.06) 等轻微差异

### 修复了什么

- 统一 key-point 颜色为 #4C8BF5
- 清理阴影风格差异

### 反思
- CSS 样式分散在多个 inline `<style>` 标签中，难以集中管理
- 建议后续考虑抽取为公共 CSS 文件
