# 有声导览服务

## 概述

游导旅游网站的有声导览服务，为用户提供专业、多语言的景点音频讲解，让旅行更加深入和便捷。

## 功能特点

- 🎧 **专业音频讲解** - 涵盖东京、大阪、首尔、曼谷4座城市
- 🌐 **多语言支持** - 中文、英语、日语、韩语、泰语
- 📥 **离线下载** - 提前下载，无网也能听
- 📍 **GPS自动播放** - 到达景点自动触发讲解
- ❤️ **收藏管理** - 收藏喜欢的导览，随时收听
- 📊 **播放进度记忆** - 自动保存播放位置

## 文件结构

```
audio-guide/
├── index.html      # 导览首页 - 热门导览、城市筛选、搜索

├── map.html        # 导览地图 - 地图标记、路线推荐
├── my-guides.html  # 我的导览 - 收藏、下载、历史
└── download.html   # 离线下载 - 批量下载、存储管理

css/
└── audio-guide.css # 有声导览专属样式

data/
└── audio-guide.json # 导览数据
```

## 使用说明

### 首页浏览
1. 在首页查看热门导览推荐
2. 按城市筛选感兴趣的导览
3. 使用搜索功能快速找到目标景点

### 开始导览
1. 点击导览卡片进入详情页
2. 选择语言（支持5种语言）
3. 点击播放按钮开始讲解
4. 可切换段落或调节播放速度

### 离线下载
1. 进入导览详情页
2. 点击「下载音频」按钮
3. 在「我的导览→已下载」查看
4. 支持一键批量下载整个城市

### 地图导览
1. 进入导览地图页面
2. 选择目标城市
3. 点击地图标记查看导览点
4. 开始导览或查看路线推荐

## 技术实现

- **前端框架**: Tailwind CSS
- **地图服务**: Leaflet + OpenStreetMap
- **音频处理**: HTML5 Audio API
- **数据存储**: LocalStorage
- **响应式设计**: 适配手机、平板、桌面

## 注意事项

1. 建议在Wi-Fi环境下下载导览，节省流量
2. 部分导览需要位置权限以启用GPS自动播放
3. 离线内容会在30天后过期，请定期更新
4. 存储空间有限，建议定期清理不需要的下载
