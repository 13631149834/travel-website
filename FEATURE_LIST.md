## 管理后台

### 管理员面板

| 页面 | 文件名 | 功能描述 |
|------|--------|----------|
| 管理主面板 | `admin.html` | 数据概览和快捷入口 |
| 用户管理 | `admin-users.html` | 用户账号管理 |
| 导游管理 | `admin-guides.html` | 导游审核和管理 |
| 评价管理 | `admin-reviews.html` | 评价审核和处理 |
| 数据备份 | `data-backup.html` | 数据库备份 |

### 管理员功能说明

#### 数据概览
- 实时用户数量统计
- 导游数量和活跃度
- 订单数量和金额
- 近 7 天/30 天趋势图
- 待处理事项提醒

#### 用户管理
- 用户列表查询
- 用户详情查看
- 账号禁用/启用
- 操作日志查看

#### 导游管理
- 入驻申请审核
- 导游资料编辑
- 认证状态管理
- 投诉记录处理

#### 评价管理
- 评价内容审核
- 敏感词过滤
- 违规评价处理
- 评价数据导出

## 技术特性

### 前端技术
- HTML5 语义化标签
- CSS3 响应式设计
- Vanilla JavaScript
- 模块化 CSS 架构

### 后端技术
- Supabase 身份认证
- PostgreSQL 数据库
- 实时数据订阅
- 文件存储服务

### 安全特性
- Row Level Security (RLS)
- 密码加密存储
- XSS 防护
- CSRF 令牌
