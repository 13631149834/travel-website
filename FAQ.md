## 账号相关

### Q1: 如何注册账号？

**A:** 
1. 点击首页右上角「注册」
2. 选择账号类型（游客/导游）
3. 填写手机号或邮箱
4. 设置登录密码
5. 获取验证码并完成验证
6. 注册成功

### Q2: 忘记密码怎么办？

**A:** 
1. 点击登录页「忘记密码」
2. 输入注册邮箱/手机号
3. 点击「发送重置链接」
4. 查收邮件/短信
5. 点击链接设置新密码

### Q3: 如何修改个人信息？

**A:** 
1. 登录后进入「我的」
2. 点击「个人中心」
3. 点击「编辑资料」
4. 修改头像、昵称、简介等信息
5. 点击「保存」

### Q4: 如何注销账号？

**A:** 
1. 进入「个人中心」→「账号安全」
2. 点击「注销账号」
3. 确认注销须知
4. 身份验证
5. 完成注销

**注意**：
- 注销后所有数据无法恢复
- 进行中的订单需先取消
- 账户余额需提前提现

### Q5: 收不到验证码怎么办？

**A:** 请按以下步骤排查：
1. 检查手机信号和网络
2. 确认手机号是否正确
3. 查看是否在黑名单中
4. 尝试重新获取
5. 稍后再次尝试

## 游客使用

### Q6: 如何搜索导游？

**A:** 
1. 进入「导游搜索」页面
2. 使用筛选条件：
   - 城市/地区
   - 语言能力
   - 服务评分
   - 价格区间
3. 浏览导游列表
4. 点击查看详情

### Q7: 如何预约导游？

**A:** 
1. 进入导游详情页
2. 选择服务类型和日期
3. 填写联系方式和需求
4. 确认订单信息
5. 提交预约请求
6. 等待导游确认

### Q8: 预约后可以取消吗？

**A:** 
- **未确认前**：可以随时取消
- **已确认后**：需提前 24 小时取消
- **当天取消**：需支付违约金
- 恶意取消可能影响账号信誉

### Q9: 如何对导游评价？

**A:** 
1. 进入「我的订单」
2. 找到已完成的订单
3. 点击「立即评价」
4. 选择评分（1-5 星）
5. 填写评价内容
6. 上传实拍图片（可选）
7. 点击「提交」

### Q10: 收藏的导游在哪里查看？

**A:** 
1. 进入「我的」→「我的收藏」
2. 可以查看所有收藏
3. 支持按类型筛选
4. 可以一键发起预约

### Q11: 如何规划旅行行程？

**A:** 
1. 进入「行程规划」
2. 选择目的地城市
3. 添加想去的景点
4. 设置每日行程安排
5. 生成行程单
6. 可以分享或导出

## 导游入驻

### Q12: 申请导游需要什么条件？

**A:** 基本条件：
- 年满 18 周岁
- 持有有效导游证
- 无不良从业记录
- 具备良好的服务意识

### Q13: 申请材料有哪些？

**A:** 需要提交：
1. 身份证正反面照片
2. 导游资格证照片
3. 个人形象照（生活照）
4. 擅长语种证明
5. 从业经历简介

### Q14: 申请审核需要多久？

**A:** 
- 审核周期：3-5 个工作日
- 材料不全将延长审核
- 可在「我的申请」查看进度

### Q15: 审核被拒绝怎么办？

**A:** 
1. 登录后查看拒绝原因
2. 根据原因补充/修改材料
3. 重新提交申请
4. 如有异议可联系客服申诉

### Q16: 导游如何设置可预约时间？

**A:** 
1. 进入「日程管理」
2. 选择日期
3. 设置每日可预约时段
4. 设置每次服务时长
5. 保存设置

### Q17: 导游如何提现收入？

**A:** 
1. 进入「财务管理」
2. 查看可用余额
3. 点击「提现」
4. 选择提现方式
5. 输入提现金额
6. 确认提交

**提现规则**：
- 最少提现金额：100 元
- 提现手续费：1%
- 到账时间：1-3 个工作日

## 订单问题

### Q18: 订单状态说明

| 状态 | 含义 | 可操作 |
|------|------|--------|
| 待确认 | 等待导游确认 | 取消 |
| 已确认 | 导游已接受 | 取消（可能扣费） |
| 进行中 | 服务已开始 | 完成确认 |
| 已完成 | 服务已结束 | 评价 |
| 已取消 | 订单已取消 | 查看 |
| 已退款 | 款项已退还 | 查看 |

### Q19: 订单有争议怎么办？

**A:** 
1. 进入订单详情
2. 点击「申请仲裁」
3. 填写争议说明
4. 上传相关证据
5. 等待平台仲裁

### Q20: 导游取消订单怎么办？

**A:** 
1. 系统自动通知
2. 款项自动退回
3. 导游需支付违约金
4. 可选择其他导游
5. 如有损失可申诉

## 支付问题

### Q21: 支持哪些支付方式？

**A:** 
| 支付方式 | 说明 |
|----------|------|
| 微信支付 | 扫码支付 |
| 支付宝 | 扫码支付 |
| 银行卡 | 信用卡/借记卡 |
| 平台余额 | 余额支付 |

### Q22: 支付失败怎么办？

**A:** 
1. 检查网络连接
2. 确认支付账户余额
3. 核实银行卡限额
4. 更换支付方式重试
5. 联系客服协助

### Q23: 如何申请退款？

**A:** 
- **未服务前**：可全额退款
- **服务已开始**：按比例退款
- **特殊情况**：联系客服处理

### Q24: 退款多久到账？

**A:** 
- 原路退回：3-7 个工作日
- 平台余额：即时到账
- 特殊情况：最长 15 个工作日

## 技术问题

### Q25: 页面加载缓慢怎么办？

**A:** 
1. 检查网络连接
2. 清除浏览器缓存
3. 更换浏览器尝试
4. 关闭不必要的后台程序
5. 切换网络（WiFi/4G）

### Q26: 图片无法显示怎么办？

**A:** 
1. 刷新页面重试
2. 清除浏览器缓存
3. 检查是否为支持的格式
4. 尝试更换浏览器
5. 联系客服反馈

### Q27: 如何清除浏览器缓存？

**A:** 各浏览器操作：

**Chrome**：
1. `Ctrl + Shift + Delete`
2. 选择时间范围
3. 勾选「缓存图片和文件」
4. 点击「清除数据」

**Firefox**：
1. `Ctrl + Shift + Delete`
2. 选择时间范围
3. 勾选「缓存」
4. 点击「立即清除」

### Q28: 如何开启 JavaScript？

**A:** 
**Chrome**：
1. 设置 → 高级 → 隐私和安全
2. 站点设置 → JavaScript
3. 开启「允许」

**Safari**：
1. 偏好设置 → 安全性
2. 勾选「启用 JavaScript」

### Q29: 遇到系统错误怎么办？

**A:** 
1. 记录错误信息
2. 截图保存
3. 刷新页面重试
4. 清除缓存后重试
5. 联系客服反馈
