## 快速入门

### 首次访问

1. 打开网站首页
2. 点击右上角「注册」创建账号
3. 填写基本信息完成注册
4. 根据需求选择「我是游客」或「我是导游」

### 账号类型说明

| 类型 | 权限说明 |
|------|---------|
| 游客 | 搜索导游、预约服务、评价反馈、收藏管理 |
| 导游 | 个人主页、订单管理、知识库、统计分析 |
| 管理员 | 全站用户管理、导游审核、内容管理 |

## 游客功能

### 搜索导游

**入口**：首页 → 导游搜索

**操作步骤**：
1. 进入导游列表页面
2. 使用筛选条件缩小范围：
   - 城市/地区
   - 语言能力
   - 服务评分
   - 价格区间
3. 点击导游卡片查看详情
4. 点击「收藏」或「立即预约」

### 预约导游服务

**入口**：导游详情页 → 预约按钮

**操作步骤**：
1. 选择服务类型（按小时/按天/包团）
2. 选择旅行日期和时间
3. 填写联系方式和特殊需求
4. 确认订单信息
5. 提交预约

**预约状态说明**：

| 状态 | 含义 |
|------|------|
| 待确认 | 等待导游确认 |
| 已确认 | 导游已接受预约 |
| 进行中 | 服务已开始 |
| 已完成 | 服务已完成 |
| 已取消 | 订单已取消 |

### 管理订单

**入口**：我的 → 我的订单

**功能说明**：
- 查看所有订单历史
- 筛选订单状态
- 查看订单详情
- 取消未确认的订单
- 对已完成订单进行评价

### 提交评价

**入口**：订单详情 → 评价按钮

**评价规则**：
- 仅可对已完成订单评价
- 评分 1-5 星
- 上传实拍图片（最多 9 张）
- 文字评价不少于 10 字

### 收藏管理

**入口**：我的 → 我的收藏

**功能说明**：
- 收藏导游（方便下次快速预约）
- 收藏精品路线
- 创建收藏夹分类管理
- 一键分享收藏内容

## 导游功能

### 申请成为导游

**入口**：成为导游 → 导游申请

**申请条件**：
- 年满 18 周岁
- 持有有效导游证
- 具备良好的服务意识
- 无不良从业记录

**申请材料**：
1. 身份证正反面照片
2. 导游资格证
3. 个人形象照（生活照）
4. 擅长语种证明
5. 从业经历简介

**审核周期**：3-5 个工作日

### 导游工作台

**入口**：导游端 → 工作台

**功能模块**：

| 模块 | 功能 |
|------|------|
| 今日概览 | 待处理订单、明日行程、收入统计 |
| 订单管理 | 接收/处理预约、查看订单详情 |
| 日程管理 | 设置可预约时间、管理团期 |
| 收入统计 | 查看收入报表、提现记录 |

### 设置可预约时间

**入口**：导游端 → 日程管理

**操作步骤**：
1. 进入日程管理页面
2. 选择需要设置的日期
3. 设置每日可预约时段
4. 设置每次服务时长
5. 保存设置

### 路线规划

**入口**：导游端 → 路线规划

**功能说明**：
- 创建自定义路线模板
- 添加景点、餐厅、休息点
- 设置每个点停留时间
- 预估路线总时长
- 导出路线 PDF

### 景点知识库

**入口**：导游端 → 知识库

**内容分类**：
- 景点介绍
- 历史背景
- 解说词参考
- 常见问答
- 实用贴士

### 财务管理

**入口**：导游端 → 财务管理

**功能说明**：
- 查看收入明细
- 月度收入报表
- 提现申请
- 发票管理
- 收入趋势分析

## 管理后台

### 访问管理后台

**入口**：`/admin.html`

**登录方式**：
- 使用管理员账号登录
- 默认管理员：`admin@youdao-travel.com`

### 数据概览

**入口**：管理后台 → 仪表盘

**展示内容**：
- 今日新增用户/导游/订单
- 平台总用户数、导游数、订单数
- 近 7 天/30 天趋势图
- 待处理事项提醒

### 用户管理

**入口**：管理后台 → 用户管理

**功能说明**：
- 查看所有用户列表
- 搜索指定用户
- 查看用户详情
- 禁用/启用用户
- 查看用户操作日志

### 导游管理

**入口**：管理后台 → 导游管理

**审核流程**：
1. 查看导游申请列表
2. 审核申请材料
3. 通过/拒绝申请
4. 认证通过的导游显示「金牌导游」标识

**管理操作**：
- 查看导游详情
- 修改导游状态
- 设置导游等级
- 查看导游收入
- 下架违规导游

### 评价管理

**入口**：管理后台 → 评价管理

**功能说明**：
- 查看所有用户评价
- 审核敏感内容
- 回复/删除违规评价
- 导出评价数据

## 常用操作

### 修改个人信息

**入口**：我的 → 个人中心 → 编辑资料

**可修改项**：
- 头像
- 昵称
- 手机号
- 个人简介
- 密码

### 找回密码

**入口**：登录页 → 忘记密码

**操作步骤**：
1. 输入注册邮箱
2. 点击发送重置链接
3. 查收邮件并点击链接
4. 设置新密码

### 绑定手机号

**入口**：个人中心 → 账号安全 → 绑定手机

**操作步骤**：
1. 输入手机号
2. 获取验证码
3. 输入验证码完成绑定

### 消息通知

**入口**：消息中心

**通知类型**：
- 订单状态更新
- 导游回复通知
- 系统公告
- 活动提醒

**设置说明**：
- 可设置是否接收邮件通知
- 可设置消息免打扰时段

## 安全提示

1. **保护账号安全**
   - 不要在公共电脑保存密码
   - 定期更换密码
   - 开启两步验证

2. **谨防诈骗**
   - 不要私下转账
   - 确认导游身份后再预约
   - 遇到可疑情况及时举报

3. **隐私保护**
   - 不要随意透露个人信息
   - 谨慎授权第三方应用
   - 注销账号时清除所有数据
