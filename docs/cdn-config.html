🚀 什么是CDN？

CDN (Content Delivery Network) 即内容分发网络，通过在全球部署的边缘服务器， 将网站静态资源缓存到离用户最近的节点，从而实现：

⚡
加速访问
就近访问，减少延迟
🛡️
提升安全
抵御DDoS攻击
💰
降低成本
减少源站带宽

🔵 Cloudflare 免费版配置（推荐）

1 注册并添加网站

  1. 访问 cloudflare.com 注册账号
  2. 点击 "Add a Site" 添加您的域名
  3. 选择 Free 套餐
  4. 系统会自动扫描现有DNS记录

2 配置DNS 

类型    名称    内容              代理状态
A       @      你的服务器IP       ☁️ 已代理
CNAME   www    yourdomain.com     ☁️ 已代理

💡 提示：将代理状态设为"已代理"，Cloudflare才能加速您的网站

3 更新域名服务器

  1. 在Cloudflare获取分配的Nameservers地址
  2. 登录您的域名注册商（如阿里云、腾讯云）
  3. 将域名的NS记录替换为Cloudflare提供的地址
  4. 等待DNS传播（通常5分钟-24小时）

4 优化设置

// 自动压缩
Speed → Optimization
→ Auto Minify: ✅ JS, CSS, HTML
→ Rocket Loader: ✅
// 缓存策略
Caching → Configuration
→ Caching Level: Standard
→ Browser Cache TTL: 4 hours

⚡ Vercel 部署配置

快速部署

# 安装Vercel CLI
npm i -g vercel

# 登录
vercel login

# 部署（在你的项目目录执行）
vercel

vercel.json 配置

{
  "version": 2,
  "routes": [
    { "src": "/(.*)", "dest": "/$1" }
  ],
  "headers": [
    {
      "source": "/(.*)",
      "headers": [
        {
          "key": "Cache-Control",
          "value": "public, max-age=31536000, immutable"
        },
        {
          "key": "X-Content-Type-Options",
          "value": "nosniff"
        }
      ]
    },
    {
      "source": "/static/(.*)",
      "headers": [
        {
          "key": "Cache-Control",
          "value": "public, max-age=31536000, immutable"
        }
      ]
    }
  ]
}
