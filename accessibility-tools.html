开启后将减少或关闭页面中的动画和过渡效果，适合对运动敏感的用户。

⌨️ 键盘导航指南

我们的网站支持完整的键盘导航。您可以使用以下快捷键在页面中进行导航和操作：

常用快捷键

Tab 移动到下一个可聚焦元素（链接、按钮等）
Shift + Tab 移动到上一个可聚焦元素
Enter 激活链接或提交表单
Space 滚动页面或激活按钮
Esc 关闭弹窗、下拉菜单或取消操作
↑ ↓ 在菜单或列表中上下移动
← → 在标签页之间切换

💡 提示：按下 ? 键可以打开快捷键帮助菜单（如果可用）。

📖 屏幕阅读器使用指南

我们的网站针对屏幕阅读器进行了优化。以下是常见屏幕阅读器的基本使用方法：

🪟 NVDA（Windows 免费开源）

  • NVDA + 空格键 - 切换“浏览模式”和“焦点模式”
  • NVDA + N - 打开 NVDA 菜单
  • Tab / Shift + Tab - 浏览页面元素
  • H / Shift + H - 跳转到下一个/上一个标题
  • D / Shift + D - 跳转到下一个/上一个地标区域

下载 NVDA （在新窗口中打开）

🍎 VoiceOver（macOS / iOS 内置）

  • Command + F5 - 开启/关闭 VoiceOver（macOS）
  • VO + 方向键 - 导航和交互（VO = VoiceOver 键，通常是 Control + Option）
  • VO + Shift + M - 打开转子菜单
  • VO + H / Shift + H - 跳转到下一个/上一个标题
  • VO + Tab - 在控件之间移动

了解更多 VoiceOver （在新窗口中打开）

👓 JAWS（Windows 商业软件）

  • Alt + 冒号 (:) - 打开 JAWS 菜单
  • Tab / Shift + Tab - 浏览页面元素
  • H / Shift + H - 跳转到下一个/上一个标题
  • R / Shift + R - 阅读页面
  • Insert + F6 - 显示页面标题列表

📱 TalkBack（Android 内置）

  • 音量键 - 开启/关闭 TalkBack
  • 滑动 - 单指滑动浏览元素
  • 点击 - 单指点击激活元素
  • 双指 - 双指滚动列表
  • 上/下/swipe - 在区域之间导航

了解更多 TalkBack （在新窗口中打开）

🔍 浏览器缩放与缩放级别

您可以通过调整浏览器缩放级别来放大或缩小页面内容。我们建议使用 100% - 200% 的缩放级别以获得最佳体验。

常用浏览器缩放快捷键

Ctrl 和 + 放大页面（PC）
Ctrl 和 - 缩小页面（PC）
Ctrl 和 0 重置为默认缩放级别（PC）
Command 和 + 放大页面（Mac）
Command 和 - 缩小页面（Mac）
Command 和 0 重置为默认缩放级别（Mac）

您也可以在浏览器设置中更改默认缩放级别，或者使用浏览器内置的辅助功能设置。
