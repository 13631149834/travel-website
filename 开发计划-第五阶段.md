### 任务2：数据库表设计

#### 用户表 (users)
```sql
- id: uuid (主键)
- email: string
- role: string (guide/tourist)
- nickname: string
- avatar: string
- created_at: timestamp
```

#### 导游资料表 (guide_profiles)
```sql
- id: uuid
- user_id: uuid (外键)
- name: string
- phone: string
- license_no: string
- years: integer
- languages: string[]
- specialties: string[]
- bio: text
- verified: boolean
```

#### 评价表 (reviews)
```sql
- id: uuid
- guide_id: uuid
- tourist_name: string
- rating: integer
- comment: text
- created_at: timestamp
```

#### 学习记录表 (learning_records)
```sql
- id: uuid
- user_id: uuid
- question_id: integer
- is_correct: boolean
- answered_at: timestamp
```
