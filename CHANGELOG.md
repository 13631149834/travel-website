### v2.0.0 (2024-03-15) - 主要版本更新

#### ✨ 新增功能

| 模块 | 功能 |
|------|------|
| **Supabase 集成** | 用户认证、数据存储、实时订阅 |
| **管理后台** | 用户管理、导游审核、评价管理 |
| **游客端** | 完整的订单、收藏、评价流程 |
| **导游端** | 入驻申请、工作台、财务管理 |

#### 🔄 技术架构升级

```
旧架构：
HTML + localStorage

新架构：
HTML + Supabase (PostgreSQL + Auth + Storage)
```

## 升级指南

### 从 v1.x 升级到 v2.0

1. **数据迁移**
   - 导出原有 localStorage 数据
   - 在 Supabase 中导入用户数据
   - 迁移订单和收藏记录

2. **配置更新**
   - 创建 Supabase 项目
   - 执行数据库 Schema
   - 更新前端配置文件

3. **测试验证**
   - 测试用户注册登录
   - 测试订单流程
   - 测试管理功能

详细升级指南请参考：[SUPABASE_GUIDE.md](./SUPABASE_GUIDE.md)
